Morse code is a method of encoding text characters into sequences of short and long signals, commonly known as “dots” and “dashes.
The system was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail during the rise of telegraph technology.
The invention of the electric telegraph created a need for a simple coding system that could convert letters and numbers into signals that could travel through wire.
Morse code works by assigning each letter, number, and some punctuation marks a unique sequence of dots and dashes. A dot represents a short signal, while a dash represents a longer signal.
